# MScFE690_Capstone-Project_ML-Application-to-Long-Short-Pair-Trading-Strategies

**Team Members:**
- Anne-Marie Antoinette ATIGNON
- Samuel Enyan
- Pedaproiu Venkata Sesa Sai Likhith
  
A systematic, market-neutral equity pairs trading strategy using regime-aware machine learning (LightGBM, XGBoost, and LSTM) to generate signals and manage risk. Compares ML-based signals against a static cointegration benchmark (Engle-Granger/Johansen) using walk-forward cross-validation with transaction costs and Fama-French 5-Factor attribution.
